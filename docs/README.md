# Devin.ai AutoReg 🚀

Автоматическая регистрация аккаунтов Devin AI с пробными подписками и удобным менеджером сессий.

**Версия 3.0.0** — с авто-генерацией карт, ротацией и статистикой BIN-ов

## ✨ Новые фичи (v3.0.0)

- 🔍 **BIN-генератор** — база 148 BIN-ов из 12 стран с интерактивным меню
- 🤖 **Авто-генерация карт** — срока, CVC, длины карты (не нужно вручную)
- 📊 **Статистика BIN-ов** — запоминает рабочие BIN-ы и приоритизирует их
- 🔄 **Ротация биллинга** — разные адреса для каждого аккаунта (25 стран)
- 🌐 **Ротация прокси** — разные прокси из списка для каждого аккаунта
- 🖥️ **Fingerprint ротация** — разные GPU/viewport для каждого аккаунта
- ⏱️ **Авто-тайминги** — подстраивает задержки под твоё окружение
- ✅ **Валидация BIN-ов** — онлайн-проверка через binlist.net (отсеивает prepaid)

## ⚠️ Важно: VPN / Прокси

Для работы скрипта **обязательно** нужен качественный платный VPN или хорошие прокси. Без них:
- Регистрация аккаунтов может быть заблокирована по IP
- Stripe Checkout может отклонять карты или не загружаться
- hCaptcha может не решаться или требовать сложные изображения

**Рекомендации:**
- **Платный VPN** — тунели WireGuard, DurevVPN, ExpressVPN и аналоги (не бесплатные!)
- **Прокси** — residential или mobile-прокси (страна США). Datacenter-прокси часто детектятся
- Настройте прокси в `config.js`: `PROXY: 'http://user:pass@ip:port'` или запустите VPN перед скриптом

## 🛠 Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Установка Playwright Chromium

```bash
npx playwright install chromium
```

### 3. Настройка конфигурации

Откройте `config.js` и укажите свои данные:
- `BINS` — BIN-ы карт (можно несколько для ротации, [] = авто-выбор)
- `EXP_DATE`, `CVC_CODE` — срок и CVC карты (или 'auto' для авто-генерации)
- `PROXY` — прокси (или `null` для ротации из списка)
- `ACCOUNTS_COUNT` — количество аккаунтов (0 = бесконечно)

### 4. Запуск через меню (рекомендуется)

```bash
node menu.js
```

Меню позволяет:
- Настроить BIN, срок действия карты, CVC
- Указать количество аккаунтов
- Включить/выключить headless режим
- Включить/выключить ручной режим
- Управлять сессиями через менеджер сессий
- Открыть BIN-генератор
- Запустить авторегистрацию

### 5. Интерактивный запуск (новое)

```bash
node start_interactive.js
```

Интерактивная настройка с 10 вопросами и подсказками:
- Выбор BIN-ов или авто-генерация
- Настройка прокси и ротации
- Выбор количества аккаунтов
- Пресеты для сохранения настроек

### 6. BIN-генератор (новое)

```bash
node autoreger_data/internal/bin-lookup.js
```

Интерактивный BIN-генератор:
- База 148 BIN-ов из 12 стран
- Проверка конкретных BIN-ов
- Генерация случайных credit BIN-ов
- Фильтрация по стране и типу карты

### 7. Прямой запуск авторегера

```bash
node autoreger.js
```

## ⚙️ Конфигурация (config.js)

Все параметры находятся в одном файле — **`config.js`**. Не нужно редактировать другие файлы.

### Параметры карты

| Параметр | Описание | Пример |
|----------|----------|--------|
| `BINS` | BIN-ы карт (строка или массив) | `['5154620022', '429544']` |
| `EXP_DATE` | Срок действия карты (ММ/ГГ) | `'02/30'` |
| `CVC_CODE` | CVC-код (3-4 цифры) | `'599'` |
| `BILLING_NAME` | Имя для Stripe | `'John Doe'` |
| `BILLING_ADDRESS` | Адрес | `'350 5th Ave'` |
| `BILLING_CITY` | Город | `'New York'` |
| `BILLING_ZIP` | Почтовый индекс | `'10118'` |
| `BILLING_COUNTRY` | Страна | `'US'` |

### Параметры аккаунтов

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `ACCOUNT_PASSWORD` | Пароль для всех аккаунтов | `'StrongPassword_2026!!!'` |
| `ACCOUNTS_COUNT` | Количество аккаунтов (0 = ∞) | `5` |

### Параметры браузера

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `HEADLESS` | Без GUI (true/false) | `false` |
| `STEALTH_ENABLED` | Маскировка автоматизации | `true` |
| `MANUAL_MODE` | Ручной режим при ошибке оплаты | `true` |
| `FORCE_HEADLESS_WITH_MANUAL` | Headless даже в ручном режиме | `false` |

### Тайминги (в миллисекундах)

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `DELAY_BEFORE_TRIAL_CLICK` | Пауза перед нажатием триала | `[5500, 8500]` |
| `DELAY_AFTER_CODE_INPUT` | Пауза после ввода OTP | `7500` |
| `DELAY_PAYMENT_WAIT` | Макс. время ожидания оплаты | `90000` |
| `DELAY_OTP_WAIT` | Макс. время ожидания OTP | `75000` |

### Уведомления

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `SOUND_NOTIFICATIONS` | Звуковые уведомления (Windows) | `true` |
| `CAPTCHA_NOTIFICATIONS` | Расширенные уведомления о капче | `true` |

### Менеджер сессий

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `AUTO_ADD_TO_SESSION_MANAGER` | Автоматически добавлять сессии | `true` |
| `KEEP_BROWSER_OPEN` | Оставлять браузер открытым | `false` |

## 🔧 Решение проблем

### Проблема: "fetch failed"

**Причина:** Devin изменил защиту или проблемы с сетью.

**Решение:**
1. Проверьте VPN/прокси
2. Попробуйте другой BIN
3. Увеличьте тайминги в `config.js`
4. Подождите некоторое время перед следующей попыткой

### Проблема: Капча не решается

**Причина:** Слишком сложная капча или проблемы с прокси.

**Решение:**
1. Включите `MANUAL_MODE: true` в `config.js`
2. При появлении капчи решите её вручную
3. Используйте качественные residential прокси

### Проблема: Карта отклоняется

**Причина:** Недействительный BIN или проблемы с Stripe.

**Решение:**
1. Попробуйте другой BIN из списка
2. Увеличьте `BIN_MAX_RETRIES` в `config.js`
3. Проверьте данные биллинга в конфиге

### Проблема: Меню не реагирует на клавиши

**Причина:** Неправильное состояние терминала.

**Решение:**
1. Нажмите `Ctrl+C` для выхода
2. Перезапустите меню: `node menu.js`

## 🆕 Новые фичи v3.0.0

### Авто-генерация карт

При `EXP_DATE: 'auto'`, `CVC_CODE: 'auto'`, `CARD_LENGTH: 'auto'` скрипт автоматически генерирует:
- **Срок действия** — 1-3 года в будущем
- **CVC** — 3 цифры для VISA/MC, 4 для AMEX
- **Длина карты** — 16 или 15 в зависимости от BIN

Не нужно вручную указывать данные карты — всё генерируется автоматически.

### Статистика BIN-ов

Скрипт автоматически записывает статистику в `logs/known_bins.json`:
- Рабочие BIN-ы приоритизируются
- Проблемные BIN-ы попадают в cooldown (10 минут после 3+ неудач)
- При `BINS: []` используется встроенная база 148 BIN-ов

### Валидация BIN-ов

- `VALIDATE_BINS: true` — проверка типа карты перед запуском
- `VALIDATE_BINS_ONLINE: true` — онлайн-проверка через binlist.net
- Отсеивает prepaid карты на старте

### Ротация параметров

- **BILLING_ROTATION** — разные биллинговые данные для каждого аккаунта (25 стран)
- **PROXY_ROTATION** — разные прокси из списка для каждого аккаунта
- **FINGERPRINT_ROTATION** — разные GPU/viewport для каждого аккаунта (15 профилей)

Каждый аккаунт выглядит как уникальный пользователь.

### Авто-тайминги

`AUTO_TIMING: true` автоматически подбирает оптимальные задержки:
- Профиль сохраняется в `logs/timing_profile.json`
- Анализ прошлых запусков
- Скрипт сам подстраивается под твоё окружение
