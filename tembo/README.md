# Tembo Autoreger

Автоматизация регистрации аккаунтов Tembo.io

## Что даёт Tembo
- **20 токенов в день** бесплатно
- **Claude Opus 4.7** + GPT 5.5 + GLM 5.1
- Работа с GitHub репозиториями

## Процесс регистрации

1. Перейти на https://app.tembo.io/
2. Ввести: Email, Password, First Name, Last Name
3. Нажать "Далее"
4. Получить код на почту (6 цифр)
5. Ввести код
6. **Подключить GitHub** (можно один GitHub на все аккаунты)
7. Выбрать НЕ пустой репозиторий (должен быть хотя бы README)
8. Выбрать модель и работать
